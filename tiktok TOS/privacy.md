# Privacy Policy — iaganBot Clips

**Last updated:** June 10, 2026

## 1. Overview

iaganBot Clips ("the App") is a personal automation tool operated by a single developer. This policy describes how TikTok user data is handled in connection with the App.

## 2. Data Collection

The App does not collect, store, or share personal data from TikTok users. The only TikTok account data accessed is the OAuth access token of the App's owner, used solely to publish content to their own TikTok profile.

## 3. Data Usage

The access token is stored locally on the developer's personal machine and is used exclusively to authenticate API requests for posting video content. It is never transmitted to third parties.

## 4. Data Sharing

No data is shared with any third party. The App operates entirely on the developer's local machine.

## 5. Data Retention

OAuth tokens are stored locally and can be revoked at any time via TikTok's security settings at [https://www.tiktok.com/settings/](https://www.tiktok.com/settings/).

## 6. Third-Party Services

The App interacts with the TikTok API. Please refer to TikTok's own Privacy Policy for information on how TikTok handles data: [https://www.tiktok.com/legal/privacy-policy](https://www.tiktok.com/legal/privacy-policy)
